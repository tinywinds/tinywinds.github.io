升级更新 node 版本

查看版本

node -v

清除npm cache

npm cache clean -force

安装n模块

npm install -g n

不行就这样：

npm install -g n -f

升级新版本（稳定版 stable ,最新版 latest ,指定版本号v8.11.1）

n latest //一般情况升级到stable即可，不要升级到latest，Bug较多
n stable //升级到最新稳定版本

查看版本号，确认是否升级成功

node -v
